# React Playground

A personal playground built with **React + Vite** to practice, experiment, and explore modern React concepts.

## Tech Stack
- React (JSX)
- Vite
- JavaScript (ES6+)
- CSS

## 📂 Insides
- Reusable UI components
- Pages (Home, About, Contact, Product)
- Context API practice
- Layout & navbar experiments

## Purpose
This repository is used for:
- Practicing React fundamentals
- Building and testing components
- Experimenting with layouts and state management
